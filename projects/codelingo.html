CodeLingo

Project Brief

Duration: 1 week

Problem Statement: Create an app that would be useful for School of Code bootcampers.

Solution: An app that tests bootcampers' knowledge on coding languages; similar to a 'DuoLingo for code'.

App Features

  • Filter by question category
  • Score accumulates each answer
  • Correct answer is revealed each incorrect guess
  • Togglable relaxing background music
  • Animation and sound on each correct answer

What I have learnt

  • Improved coding using Javascript and React
  • Working as a team and approaching problems systematically
  • Using agile strategies to manage the project, like kanban boards and standups
  • The importance of testing to a project
  • Improved workflow and collaboration using Git and Github
  • The importance of time management and teamwork when working in an agile way
