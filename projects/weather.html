Weather App

Project Brief

Duration: 1-2 weeks

Problem Statement: Create a weather app using TypeScript and Open Weather's API

App Features

  • Real time weather information
  • Uses data from OpenWeather
  • Searchable weather anywhere in the globe
  • 3-hour weather forecast
  • Additional weather metrics like wind speed, humidity, pressure and feels-like temperature

What I have learnt

  • Better understanding of TypeScript
  • How to specify types for variables, parameters and return values
  • Improved my handling of APIs
  • More experience working with large datasets, arrays and handling JSON data
  • Gained a better understanding of using React and React Router
