Kindr

Project Brief

Duration: 5 weeks

Problem Statement: It is difficult to know how to ask for or to give help in the community.

Solution: To create an app that brings communities together and allows people to volunteer in an easier, safer and more structured way

App Features

  • User Registration & log in
  • Create, edit and delete tasks
  • Accept and unaccept tasks
  • See all tasks associated with your user profile
  • Browse all tasks
  • Filter all tasks by category
  • Update and edit user profile
  • Responsive design for mobile and desktop

What I have learnt

  • Gained a better understanding of using React and React Router
  • Improved my use of Figma to create wireframes and mockups
  • How to use Supabase to create a database and manage user registration
  • The importance of testing to a project
  • Improved workflow and collaboration using Git and Github
  • The importance of time management and teamwork when working in an agile way
  • Iterative development and the importance of setting clear goals and deadlines
